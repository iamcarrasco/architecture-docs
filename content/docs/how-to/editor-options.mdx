# Editor Options

This page covers the editor UI, toolbar, and file management features.

## Toolbar

### Formatting
- Bold / Italic / Underline
- Headings
- Lists (ordered/unordered)
- Block quote

### Content insertion
- Links
- Tables
- Code blocks
- Thematic breaks
- Image upload

### History
- Undo / Redo

## File management

### Create
- Create a new file
- Create a folder by saving a file inside it

### Search
- Filter files by name or path

### Delete
- Select files or folders and delete
- Deleting all files inside a folder removes the folder from the repo

## Status controls
- Status selector appears on the right
- Status drives the badge shown in docs

## Save behavior
- Autosave drafts while typing
- Save writes to draft branch or main depending on status

## Tips
- Use short filenames with hyphens
- Keep one topic per file

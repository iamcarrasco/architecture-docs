# Save Flow & Status Badges

## Save modes
- **Draft/PR** saves to a draft branch and may open a PR
- **Approved** saves directly to main

## Status badges
- **Draft**: work in progress
- **In Review**: PR is open
- **Approved**: merged to main
- **Decommissioned**: deprecated content

## Notes
- Approved status is only final when committed to main.
- Use "Update from main" if your draft is out of date.
